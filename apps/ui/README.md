# @pipelab/ui

The main visual automation interface for Pipelab.

## Tech Stack
- **Framework**: Vue 3
- **UI Library**: PrimeVue v4 with Tailwind/PrimeFlex
- **State Management**: Pinia
- **Communication**: WebSockets (to @pipelab/cli)

## Development
```bash
pnpm dev        # Start Vite dev server
pnpm build      # Build for production
```
