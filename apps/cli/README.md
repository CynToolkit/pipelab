# @pipelab/cli

Standalone command-line interface and WebSocket server for Pipelab.

## Features
- **Headless Execution**: Run Pipelab pipelines from the terminal.
- **WebSocket Server**: Provides a backend for the Pipelab UI.
- **Self-Contained**: Can be packaged into a standalone binary using `pkg`.

## Development
```bash
pnpm dev        # Start in watch mode
pnpm build      # Build the project
pnpm pkg        # Package into a standalone binary
```
